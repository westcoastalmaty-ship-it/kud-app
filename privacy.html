Политика конфиденциальности

Последнее обновление: 29 апреля 2026

Приложение «Куда уходят деньги» (далее — «Приложение») создано для помощи пользователям в учёте и анализе личных расходов. Мы серьёзно относимся к конфиденциальности ваших данных и подробно описываем в этой политике, какие данные собираем, как используем и защищаем.

Главное в одном предложении: мы не продаём ваши данные третьим лицам, не показываем рекламу и храним ваши финансовые данные только на вашем устройстве.

1. Какие данные собирает Приложение

1.1 Данные транзакций (хранятся локально)

Приложение распознаёт и сохраняет следующие данные о ваших расходах:

  • Сумма транзакции
  • Название мерчанта (магазин, организация)
  • Категория расхода (продукты, транспорт и т.д.)
  • Дата и время операции
  • Тип операции (покупка, перевод, оплата услуг)

Эти данные хранятся только на вашем устройстве в локальной базе данных и не передаются на наши серверы. При удалении приложения все данные удаляются вместе с ним.

1.2 Данные с экранов банковских приложений (через Accessibility Service)

С вашего разрешения Приложение использует Accessibility Service для чтения содержимого экранов банковских приложений (Kaspi, Halyk, Beeline и др.), чтобы автоматически распознавать совершаемые транзакции.

Что происходит с этими данными:

  • Текст с экрана отправляется на наш защищённый сервер для распознавания
  • Сервер возвращает результат распознавания (сумма, мерчант, категория)
  • Распознанная транзакция сохраняется локально на вашем устройстве
  • Текст с экрана не сохраняется и не передаётся третьим лицам

1.3 Данные push-уведомлений (через Notification Listener)

С вашего разрешения Приложение читает push-уведомления только от банковских приложений для распознавания транзакций. Уведомления других приложений (мессенджеры, соцсети, личная переписка) Приложение игнорирует.

1.4 Данные подписки (через Google Play Billing)

При оформлении подписки Google Play Billing передаёт нам только:

  • Идентификатор покупки (purchase token)
  • Тип подписки (PRO или BASIC)
  • Дату следующего автосписания

Данные банковской карты обрабатываются исключительно Google — мы их не видим и не храним.

2. Как мы используем данные

Собранные данные используются исключительно для:

  • Распознавания и категоризации ваших расходов
  • Построения графиков и аналитики ваших финансов
  • Расчёта прогресса по бюджету и целям накоплений
  • Отправки полезных уведомлений (итоги дня, месяца)
  • Управления подпиской
Мы НЕ:
  • Не продаём ваши данные третьим лицам
  • Не показываем рекламу в Приложении
  • Не используем данные для маркетинга
  • Не передаём данные в социальные сети или рекламные сети

3. Где и как хранятся данные

3.1 Локальные данные

Все ваши финансовые данные (транзакции, бюджет, цели) хранятся локально на вашем устройстве в зашифрованной базе данных Android. Они не покидают ваш телефон.

3.2 Облачная обработка распознавания

Для распознавания транзакций Приложение использует наш защищённый сервер на базе Cloudflare Workers, который, в свою очередь, обращается к API Anthropic Claude для семантического анализа. При этом:

  • Передаётся только обезличенный текст с экрана
  • Не передаются ваши имя, номер карты, телефон или другие идентификаторы
  • Сервер не сохраняет историю запросов после обработки (TTL кэша 30 дней)
  • Передача данных защищена шифрованием HTTPS

4. Передача данных третьим лицам

Мы передаём минимально необходимые данные следующим сервисам:

Сервис Какие данные Зачем
Cloudflare Workers Текст с экрана банка Распознавание транзакций
Anthropic Claude API Текст с экрана банка Семантический анализ
Google Play Billing Данные подписки Обработка платежей

5. Безопасность

Мы применяем следующие меры защиты:

  • Все сетевые запросы выполняются по HTTPS
  • Локальная база данных хранится в защищённой области приложения
  • API-ключи серверов хранятся в защищённом хранилище Cloudflare Secrets
  • Доступ к данным приложения ограничен системой Android

6. Ваши права

Вы имеете право:

  • Удалить данные — удалив Приложение, все ваши данные удаляются с устройства
  • Отключить автораспознавание — в настройках Android отключить Accessibility Service
  • Отменить подписку — через Google Play в любое время
  • Получить копию данных — запросив через email поддержки
  • Удалить аккаунт и данные на наших серверах — запросив через email поддержки

7. Использование Accessibility Service

Согласно политике Google Play: Accessibility Service в нашем Приложении используется исключительно для чтения экранов банковских приложений с целью распознавания финансовых транзакций. Это основная заявленная функциональность Приложения, без которой невозможна работа автоматического учёта расходов.

Конкретно Accessibility Service:

  • Активируется только при открытии банковских приложений
  • Не читает экраны других приложений (мессенджеры, браузер и т.д.)
  • Не выполняет действия за пользователя
  • Может быть отключён в настройках Android в любой момент

8. Дети

Приложение не предназначено для использования детьми младше 13 лет. Мы сознательно не собираем личные данные детей. Если вы обнаружили, что ребёнок предоставил нам свои данные, свяжитесь с нами для их удаления.

9. Изменения политики

Мы можем периодически обновлять эту политику конфиденциальности. О существенных изменениях будем уведомлять через приложение. Дата последнего обновления указана в начале документа.
