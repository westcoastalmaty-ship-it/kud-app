Privacy Policy

Last updated: April 29, 2026

The "Where Money Goes" application (hereinafter — "Application") is designed to help users track and analyze personal expenses. We take your data privacy seriously and describe in detail what data we collect, how we use it, and how we protect it.

In one sentence: we don't sell your data to third parties, don't show ads, and store your financial data only on your device.

1. What data the Application collects

1.1 Transaction data (stored locally)

The Application recognizes and saves the following expense data:

  • Transaction amount
  • Merchant name (store, organization)
  • Expense category (groceries, transport, etc.)
  • Date and time of the operation
  • Type of operation (purchase, transfer, service payment)

This data is stored only on your device in a local database and is not transmitted to our servers. When you uninstall the application, all data is deleted with it.

1.2 Bank app screen data (via Accessibility Service)

With your permission, the Application uses Accessibility Service to read the contents of bank application screens (Kaspi, Halyk, Beeline, etc.) to automatically recognize transactions you make.

What happens to this data:

  • Screen text is sent to our secure server for recognition
  • The server returns the recognition result (amount, merchant, category)
  • The recognized transaction is stored locally on your device
  • Screen text is not stored and not transferred to third parties

1.3 Push notification data (via Notification Listener)

With your permission, the Application reads push notifications only from bank applications to recognize transactions. Notifications from other applications (messengers, social networks, personal correspondence) are ignored by the Application.

1.4 Subscription data (via Google Play Billing)

When subscribing, Google Play Billing transfers to us only:

  • Purchase identifier (purchase token)
  • Subscription type (PRO or BASIC)
  • Date of next auto-renewal

Bank card data is processed exclusively by Google — we don't see or store it.

2. How we use the data

The collected data is used solely for:

  • Recognizing and categorizing your expenses
  • Building charts and analytics of your finances
  • Calculating budget progress and savings goals
  • Sending useful notifications (daily, monthly summaries)
  • Subscription management
We do NOT:
  • Sell your data to third parties
  • Show ads in the Application
  • Use data for marketing purposes
  • Share data with social networks or ad networks

3. Where and how data is stored

3.1 Local data

All your financial data (transactions, budget, goals) is stored locally on your device in an encrypted Android database. It does not leave your phone.

3.2 Cloud processing for recognition

For transaction recognition, the Application uses our secure server based on Cloudflare Workers, which in turn calls the Anthropic Claude API for semantic analysis. In doing so:

  • Only anonymized screen text is transmitted
  • Your name, card number, phone, or other identifiers are not transmitted
  • The server doesn't save request history after processing (cache TTL 30 days)
  • Data transmission is protected by HTTPS encryption

4. Third-party data sharing

Service What data Purpose
Cloudflare Workers Bank screen text Transaction recognition
Anthropic Claude API Bank screen text Semantic analysis
Google Play Billing Subscription data Payment processing

5. Security

  • All network requests are made via HTTPS
  • Local database is stored in the protected app area
  • Server API keys are stored in Cloudflare Secrets vault
  • Access to app data is restricted by the Android system

6. Your rights

  • Delete data — uninstalling the Application deletes all your data from the device
  • Disable auto-recognition — disable Accessibility Service in Android settings
  • Cancel subscription — via Google Play at any time
  • Get a copy of your data — by requesting via support email
  • Delete your data on our servers — by requesting via support email

7. Use of Accessibility Service

According to Google Play policy: Accessibility Service in our Application is used solely for reading bank application screens for the purpose of recognizing financial transactions. This is the main declared functionality of the Application, without which automatic expense tracking is impossible.
  • Activates only when banking applications are opened
  • Does not read screens of other applications
  • Does not perform actions on behalf of the user
  • Can be disabled in Android settings at any time

8. Children

The Application is not intended for use by children under 13. We do not knowingly collect personal data from children.

9. Changes to this policy

We may periodically update this privacy policy. We will notify you of significant changes through the application.
